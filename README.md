# Library Management System using Python

##  Project Purpose
This project is a Python-based Library Management System designed to manage books and users efficiently.

##  Features
- Add new books
- View available books
- Add users
- Issue books
- Return books
- File storage using text files

##  How to Run
1. Download the project files
2. Run main.py
3. Use menu options to interact

##  Files
- main.py → Main program
- book.py → Book class
- user.py → User class
- library.py → Library logic
- file_handler.py → File operations

##  Example Usage
User selects option 1 → Adds book  
User selects option 4 → Issues book
